  1. What CDN stands for?

    2. CDN significa Continous Delivery Network. Trata-se de um serviço web de entrega de dados. Esses servidores servem como intermediários entre o site e o usuário, dividindo a carga de dados entre ambos até que tudo seja entregue ao cliente.

  2. Por quê é vantajoso utilizar um CDN em detrimento de um arquivo na aplicação?

    3. Porque os dados proporcionados pela CDN são dados servidos a várias páginas, podendo ser reaproveitados pelo cache dos navegadores dos usuários, economizando tráfego no servidor da página e evitando que o os usuários precisem refazer o download multiplas vezes dos mesmos arquivos.

  3. Qual a diferença do Bundle para o js individual na adesão do bootstrap na aplicação?

    4. Caso o desenvolvedor não vá fazer uso de certos componentes mais complexos do framework, pode optar por escolher apenas o que lhe é conveniente e economizar alguns dados.
